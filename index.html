Widget Status: Look for the chat bubble in the bottom-right corner of this page.

About This Test

This is a standalone HTML page that simulates an external client website. It demonstrates how any website can integrate the InstaDialog widget with just a simple script tag.

The widget loads from: https://instadialog.com/api/widget/widget.js

Integration Code Used

<script> (function() { var script = document.createElement('script'); script.src = 'https://instadialog.com/api/widget/widget.js'; script.setAttribute('data-api-key', 'sk-inst-demo123'); script.setAttribute('data-title', 'Customer Support'); script.setAttribute('data-subtitle', 'How can we help?'); script.setAttribute('data-position', 'bottom-right'); script.setAttribute('data-primary-color', '#007bff'); document.head.appendChild(script); })(); </script>

Testing Instructions

  1. Look for the chat bubble in the bottom-right corner
  2. Click the bubble to open the chat interface
  3. Type a test message and press Enter
  4. You should receive a demo response from the AI
Note: This page can be hosted anywhere (GitHub Pages, Netlify, your own server, etc.) and the widget will still work as long as instadialog.com is accessible.
